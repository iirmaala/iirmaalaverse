# Nama Project
ElevenTwo Lighthouse

## Deskripsi
Project ini adalah aplikasi yang bermanfaat untuk pelajar karena berisi berbagai kebutuhan sekolah, seperti jadwal pelajaran, absensi, dan informasi penting lainnya. Aplikasi ini membantu siswa untuk lebih terorganisir dan memudahkan pengelolaan kegiatan sekolah.

## Struktur Folder
ElevenTwo-Lighthouse/
├─ images/             ← semua gambar (icon, sampul, background)
├─ index.html          ← halaman utama
├─ style.css           ← styling website
├─ script.js           ← kode JavaScript
├─ manifest.json       ← konfigurasi web app
├─ service-worker.js   ← service worker PWA
└─ README.md           ← file ini

## Cara Menjalankan
1. Pastikan semua file sudah lengkap, terutama folder `images/`.  
2. Buka file `index.html` di browser.  
3. Website akan langsung tampil.

## Catatan
- Semua gambar harus berada di folder `images/`.  
- Gunakan nama file gambar yang rapi (huruf kecil, tanpa spasi).  
- File `manifest.json` dan `service-worker.js` khusus untuk PWA.
